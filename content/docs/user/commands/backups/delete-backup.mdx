# Introduction

How-to guide on how to delete a backup. Wanna delete one of ur backup? Maybe it wasnt password protected or maybe you changes up something? Fear not we gotchu covered!

# Usage

This command can be used by **"backup.create"** user just by running `/backups delete`

![Image](/backup-delete.png)
