# List Lockdowns

The `/lockdowns list` command allows you to view all currently active lockdowns on your server. This is useful for quickly checking which channels or roles are currently restricted.

## Usage

```
/lockdowns list
```

![Image](/lockdowns-list.png)

## Permissions

- `lockdowns.list` â€” Allows viewing the list of active lockdowns.

## What it does

- Displays a list of all active lockdowns, including the type (full, channel, role) and the affected targets.
- Useful for server administrators to audit lockdown status at a glance.
