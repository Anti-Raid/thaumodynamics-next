# To remove AFK

run `/afk stop`

![Image](/afk-stop.png)

# Behavior Details

## Return Notification

- You'll receive a confirmation that your AFK status was removed
- Others will no longer see AFK notifications when mentioning you

# Important Notes

- Your AFK status and reason are completely cleared
- If you need to go AFK again, you'll need to use the `/afk` command again
